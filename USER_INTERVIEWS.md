# User Interviews

## Interview 1

### Name
Prabhanshu Pandey. (Computer Science Student)

### Role
Student / Beginner Developer

### Tools Used
- ChatGPT
- Gemini

### Direct Quotes

> “I mostly use free versions because I don’t use AI tools daily.”

> “Sometimes ChatGPT and Gemini give similar answers, so I switch between them.”

> “I probably wouldn’t pay for an audit tool right now.”

### Most Surprising Insight

The user frequently switched between multiple AI tools without clearly understanding which one was actually more useful for specific tasks.

### What It Changed In My Design

This reinforced the importance of:
- keeping the audit free
- showing value before asking for signup
- making recommendations simple and beginner-friendly

## Interview 2

### Name
Sachin. (Freelance Developer)

### Role
Freelance Web Developer

### Tools Used
- ChatGPT
- Gemini
- GitHub Copilot
- Blackbox AI

### Direct Quotes

> “I use several AI tools together because each one helps differently.”

> “Gemini Pro came bundled with my Jio recharge, so I use it without thinking much about pricing.”

> “I’m not always sure which AI tool gives the best value overall.”

### Most Surprising Insight

The user was paying attention to convenience and bundled access rather than directly comparing productivity value between tools.

### What It Changed In My Design

I improved the audit logic to focus more on:
- overlapping subscriptions
- duplicated functionality
- value comparisons between tools

rather than only raw pricing differences.

## Interview 3

### Name
Aman verma. (Student Developer)

### Role
Student / Coding Learner

### Tools Used
- ChatGPT
- Blackbox AI

### Direct Quotes

> “I mainly use AI tools for coding help.”

> “Most of my AI usage stays within free plans.”

> “I use multiple tools because some are better for debugging and others are better for explanations.”

### Most Surprising Insight

The user selected tools based on specific workflow strengths rather than loyalty to one provider.

### What It Changed In My Design

This influenced the recommendation engine to avoid assuming one AI platform completely replaces another.

The audit now focuses more on:
- reducing unnecessary overlap
- identifying inefficient subscriptions
- suggesting realistic optimization opportunities
