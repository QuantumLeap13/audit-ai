# AI Summary Prompt

We use OpenAI GPT-4o-mini to generate personalized AI spend audit summaries.

## Goals

- Explain optimization opportunities clearly
- Sound financially realistic
- Avoid exaggerated marketing language
- Mention annualized savings impact
- Keep summaries concise and readable

## Prompt Strategy

The prompt includes:
- Tool name
- Current spend
- Estimated savings
- User use case

The model is instructed to:
- Stay within 80–120 words
- Use professional tone
- Avoid hype
- Focus on actionable optimization insights

## Failed Experiments

Earlier prompts produced:
- overly generic responses
- unrealistic savings claims
- repetitive phrasing

Adding financial context and explicit tone constraints improved consistency significantly.
